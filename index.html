HTML Basics

HTML documents are described by HTML tags. Each tag describes how different content will be displayed.

Tags are keywords (tag names) surrounded by < > :

<tagname> content </tagname>

Tags can also have attributes, which are extra bits of information. The example below defines a link (<a> tag) to the school homepage. The href attribute indicates the link's destination.

<a href = "http://www.neiu.edu"> neiu homepage </a>
